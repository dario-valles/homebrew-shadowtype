# Homebrew tap for Shadowtype

Private, on-device AI autocomplete for macOS. → [shadowtype.app](https://shadowtype.app) · [main repo](https://github.com/dario-valles/shadowtype-mac)

## Install

```sh
brew install --cask dario-valles/shadowtype/shadowtype
```

Requires Apple Silicon (M1+) and macOS 14+. Update with `brew upgrade --cask shadowtype`.
